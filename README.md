# AI at the Edge & IIoT Class Portfolio  
A curated collection of projects demonstrating applied Artificial Intelligence, Edge Computing, and Industrial IoT systems

This portfolio showcases hands‑on work completed as part of the **AI at the Edge & IIoT** curriculum. Each project highlights real‑world problem‑solving across embedded intelligence, data pipelines, cybersecurity, autonomous systems, and cloud‑connected devices.

---

## Portfolio Overview

This repository contains multiple independent projects, each representing a core competency in Edge AI and IIoT:

| Project | Description | Skills Demonstrated |
|--------|-------------|---------------------|
| SmartGlucometerSimulation | A Python‑based IIoT smart glucometer simulator with cybersecurity planning and telemetry workflows. | Edge device simulation, Python, MQTT/HTTP telemetry, cybersecurity frameworks |
| RAG-based_Structure_for_10k_Analysis | A Retrieval‑Augmented Generation (RAG) verifier that checks numeric claims in financial analysis documents against SEC 10‑K filings. | LLMs, RAG pipelines, document parsing, structured JSON output |
| Capstone Project – Smartwatch ThingsBoard Dashboard | A full telemetry pipeline from smartwatch sensor data into ThingsBoard Cloud with dashboards, alerts, and analytics. | Edge → Cloud ingestion, dashboard design, IoT data modeling |
| Foundations of Edge Computing Labs | Coursework and lab exercises exploring edge architectures, containerization, and distributed intelligence. | Edge fundamentals, Docker, Linux, networking |
| Case Study: Autonomous Guided Vehicles (AGVs) | A research‑driven analysis of AGV systems, safety, and AI‑enabled navigation. | Technical writing, robotics concepts, safety frameworks |
| Edge AI & IoT Development Tools Report | A comparative analysis of modern tools used in edge AI development. | Research, evaluation, technical communication |
